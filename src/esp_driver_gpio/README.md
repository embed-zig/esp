# esp_driver_gpio

Sdkconfig binding for the ESP-IDF `esp_driver_gpio` component.

## ESP-IDF component

Maps to [`esp_driver_gpio`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/gpio.html).

## Module boundary

Typed sdkconfig definitions for GPIO driver IRAM placement of control functions. No runtime Zig API.

## Dependencies

No runtime dependencies.
