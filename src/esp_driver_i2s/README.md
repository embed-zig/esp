# esp_driver_i2s

Sdkconfig binding for the ESP-IDF `esp_driver_i2s` component.

## ESP-IDF component

Maps to [`esp_driver_i2s`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/i2s.html).

## Module boundary

Typed sdkconfig definitions for I2S driver: debug logging, ISR IRAM safety, and deprecation warning suppression. No runtime Zig API.

## Dependencies

No runtime dependencies.
