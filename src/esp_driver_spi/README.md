# esp_driver_spi

Sdkconfig binding for the ESP-IDF `esp_driver_spi` component.

## ESP-IDF component

Maps to [`esp_driver_spi`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/spi_master.html).

## Module boundary

Typed sdkconfig definitions for the SPI master and slave peripheral drivers. No runtime Zig API.

## Dependencies

No runtime dependencies.
