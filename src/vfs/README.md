# vfs

Sdkconfig binding for the ESP-IDF `vfs` component.

## ESP-IDF component

Maps to [`vfs`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/storage/vfs.html).

## Module boundary

Provides sdkconfig options for the Virtual Filesystem: UART console I/O, semihosting mount points, and select() support on UART/sockets. No runtime Zig API.

## Dependencies

No runtime dependencies.
