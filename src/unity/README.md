# unity

Sdkconfig binding for the ESP-IDF `unity` test framework configuration.

## ESP-IDF component

Maps to `unity` (internal test framework configuration for ESP-IDF unit tests).

## Module boundary

Provides sdkconfig options for the Unity test framework: 64-bit support, float/double precision, colored output, and backtrace on failure. No runtime Zig API.

## Dependencies

No runtime dependencies.
