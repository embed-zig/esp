# esp_gdbstub

Sdkconfig binding for the ESP-IDF `esp_gdbstub` component.

## ESP-IDF component

Maps to [`esp_gdbstub`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/gdbstub.html).

## Module boundary

Typed sdkconfig definitions for the GDB stub debugger, including task inspection support and max task limits. No runtime Zig API.

## Dependencies

No runtime dependencies.
