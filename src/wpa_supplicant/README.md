# wpa_supplicant

Sdkconfig binding for the ESP-IDF `wpa_supplicant` component.

## ESP-IDF component

Maps to `wpa_supplicant` (Wi-Fi security, part of the ESP-IDF Wi-Fi stack).

## Module boundary

Provides sdkconfig options for the WPA supplicant: 802.11k/v/r support, DPP (Easy Connect), enterprise authentication (EAP), WPA3-SAE, OWE, MBO, and debug logging. No runtime Zig API.

## Dependencies

No runtime dependencies.
