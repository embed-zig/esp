# esp_psram

Sdkconfig binding for the ESP-IDF `esp_psram` component.

## ESP-IDF component

Maps to [`esp_psram`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/).

## Module boundary

Provides a single sdkconfig flag (`CONFIG_SPIRAM`) to enable or disable external PSRAM (SPI RAM) support.

## Dependencies

No runtime dependencies.
