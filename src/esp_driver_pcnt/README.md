# esp_driver_pcnt

Sdkconfig binding for the ESP-IDF `esp_driver_pcnt` component.

## ESP-IDF component

Maps to [`esp_driver_pcnt`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/pcnt.html).

## Module boundary

Typed sdkconfig definitions for Pulse Counter: IRAM placement of control functions, ISR safety, debug logging, and deprecation warning suppression. No runtime Zig API.

## Dependencies

No runtime dependencies.
