# esp_https_server

Sdkconfig binding for the ESP-IDF `esp_https_server` component.

## ESP-IDF component

Maps to [`esp_https_server`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/protocols/esp_https_server.html).

## Module boundary

Typed sdkconfig definitions for the HTTPS server, including enable flag and event post timeout. No runtime Zig API.

## Dependencies

No runtime dependencies.
