# console

Sdkconfig binding for the ESP-IDF `console` component.

## ESP-IDF component

Maps to [`console`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/console.html).

## Module boundary

Typed sdkconfig definitions for UART console selection (default/custom/none), baudrate, port number, and help sorting. No runtime Zig API.

## Dependencies

No runtime dependencies.
