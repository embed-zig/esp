# esp_driver_gdma

Sdkconfig binding for the ESP-IDF `esp_driver_gdma` component.

## ESP-IDF component

Maps to [`esp_driver_gdma`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/gdma.html).

## Module boundary

Typed sdkconfig definitions for General DMA controller: IRAM placement of control functions, ISR safety, and debug logging. No runtime Zig API.

## Dependencies

No runtime dependencies.
