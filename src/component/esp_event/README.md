# esp_event

Sdkconfig binding for the ESP-IDF `esp_event` component.

## ESP-IDF component

Maps to [`esp_event`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/esp_event.html).

## Module boundary

Typed sdkconfig definitions for the event loop library, including loop profiling, ISR posting, and system event queue/task sizing. No runtime Zig API.

## Dependencies

No runtime dependencies.
