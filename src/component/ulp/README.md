# ulp

Sdkconfig binding for the ESP-IDF `ulp` component.

## ESP-IDF component

Maps to [`ulp`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/ulp.html).

## Module boundary

Provides sdkconfig options to enable/disable the Ultra Low Power (ULP) coprocessor. No runtime Zig API.

## Dependencies

No runtime dependencies.
