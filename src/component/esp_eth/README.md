# esp_eth

Sdkconfig binding for the ESP-IDF `esp_eth` component.

## ESP-IDF component

Maps to [`esp_eth`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/network/esp_eth.html).

## Module boundary

Typed sdkconfig definitions for Ethernet support, including internal MAC, SPI-based PHY chips (DM9051, KSZ8851SNL, W5500), and OpenETH. No runtime Zig API.

## Dependencies

No runtime dependencies.
