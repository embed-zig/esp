# esp_https_ota

Sdkconfig binding for the ESP-IDF `esp_https_ota` component.

## ESP-IDF component

Maps to [`esp_https_ota`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/protocols/esp_https_ota.html).

## Module boundary

Typed sdkconfig definitions for HTTPS OTA firmware updates, including HTTP fallback, decryption callback, and event post timeout. No runtime Zig API.

## Dependencies

No runtime dependencies.
