# esp_driver_twai

Sdkconfig binding for the ESP-IDF `esp_driver_twai` component.

## ESP-IDF component

Maps to [`esp_driver_twai`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/twai.html).

## Module boundary

Typed sdkconfig definitions for the Two-Wire Automotive Interface (TWAI/CAN) driver. No runtime Zig API.

## Dependencies

No runtime dependencies.
