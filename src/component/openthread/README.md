# openthread

Sdkconfig binding for the ESP-IDF `openthread` component.

## ESP-IDF component

Maps to [`openthread`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/network/openthread.html).

## Module boundary

Provides sdkconfig options to enable/disable OpenThread mesh networking support and DNS64 client. No runtime Zig API.

## Dependencies

No runtime dependencies.
