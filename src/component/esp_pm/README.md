# esp_pm

Sdkconfig binding for the ESP-IDF `esp_pm` component.

## ESP-IDF component

Maps to [`esp_pm`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/power_management.html).

## Module boundary

Provides sdkconfig bindings for power management and sleep configuration. Covers deep/light sleep wakeup delays, flash leakage workarounds, GPIO reset behavior, CPU power-down policy, and IRAM optimization during sleep.

## Dependencies

No runtime dependencies.
