# esp_mm

Sdkconfig binding for the ESP-IDF `esp_mm` component.

## ESP-IDF component

Maps to [`esp_mm`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/mm.html).

## Module boundary

Provides sdkconfig bindings for MMU page mode and page size configuration. Controls memory management unit page granularity settings forwarded to ESP-IDF.

## Dependencies

No runtime dependencies.
