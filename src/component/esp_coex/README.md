# esp_coex

Sdkconfig binding for the ESP-IDF `esp_coex` component.

## ESP-IDF component

Maps to [`esp_coex`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/network/esp_coex.html).

## Module boundary

Typed sdkconfig definitions for Wi-Fi/Bluetooth coexistence: enable/disable, external coexist support, and GPIO debug. No runtime Zig API.

## Dependencies

No runtime dependencies.
