# esp_http_client

Sdkconfig binding for the ESP-IDF `esp_http_client` component.

## ESP-IDF component

Maps to [`esp_http_client`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/protocols/esp_http_client.html).

## Module boundary

Typed sdkconfig definitions for the HTTP client, including authentication modes, HTTPS support, custom transport, and event post timeout. No runtime Zig API.

## Dependencies

No runtime dependencies.
