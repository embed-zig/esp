# esp_driver_rmt

Sdkconfig binding for the ESP-IDF `esp_driver_rmt` component.

## ESP-IDF component

Maps to [`esp_driver_rmt`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/rmt.html).

## Module boundary

Typed sdkconfig definitions for Remote Control Transceiver: ISR IRAM safety, receive function IRAM placement, debug logging, and deprecation warning suppression. No runtime Zig API.

## Dependencies

No runtime dependencies.
