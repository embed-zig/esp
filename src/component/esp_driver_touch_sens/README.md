# esp_driver_touch_sens

Sdkconfig binding for the ESP-IDF `esp_driver_touch_sens` component.

## ESP-IDF component

Maps to [`esp_driver_touch_sens`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/touch_element.html).

## Module boundary

Typed sdkconfig definitions for the capacitive touch sensor driver. No runtime Zig API.

## Dependencies

No runtime dependencies.
