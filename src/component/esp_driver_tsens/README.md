# esp_driver_tsens

Sdkconfig binding for the ESP-IDF `esp_driver_tsens` component.

## ESP-IDF component

Maps to [`esp_driver_tsens`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/temp_sensor.html).

## Module boundary

Typed sdkconfig definitions for the internal temperature sensor driver. No runtime Zig API.

## Dependencies

No runtime dependencies.
