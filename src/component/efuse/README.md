# efuse

Sdkconfig binding for the ESP-IDF `efuse` component.

## ESP-IDF component

Maps to [`efuse`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/efuse.html).

## Module boundary

Typed sdkconfig definitions for eFuse block length, custom table support, and virtual eFuse mode. No runtime Zig API.

## Dependencies

No runtime dependencies.
