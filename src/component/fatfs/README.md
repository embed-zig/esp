# fatfs

Sdkconfig binding for the ESP-IDF `fatfs` component.

## ESP-IDF component

Maps to [`fatfs`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/storage/fatfs.html).

## Module boundary

Provides sdkconfig bindings for the FAT filesystem implementation. Covers codepage selection, long filename support, sector size, file locking, cache policy, volume count, string function mode, and fsync behavior.

## Dependencies

No runtime dependencies.
