# wear_levelling

Sdkconfig binding for the ESP-IDF `wear_levelling` component.

## ESP-IDF component

Maps to [`wear_levelling`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/storage/wear-levelling.html).

## Module boundary

Provides sdkconfig options for the flash wear-levelling layer: sector size and performance/safety mode selection. No runtime Zig API.

## Dependencies

No runtime dependencies.
