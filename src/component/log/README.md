# log

Sdkconfig binding for the ESP-IDF `log` component.

## ESP-IDF component

Maps to [`log`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/log.html).

## Module boundary

Provides sdkconfig options for the ESP-IDF logging system: bootloader/default/maximum log levels, timestamp source, dynamic level control, tag-level cache, and color output. No runtime Zig API.

## Dependencies

No runtime dependencies.
