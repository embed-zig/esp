# esp_timer

Sdkconfig binding for the ESP-IDF `esp_timer` component.

## ESP-IDF component

Maps to [`esp_timer`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/esp_timer.html).

## Module boundary

Provides sdkconfig bindings for the high-resolution timer subsystem. Covers timer implementation backend (systimer), interrupt level, ISR/task affinity, task stack size, profiling, and legacy timer task settings.

## Dependencies

No runtime dependencies.
