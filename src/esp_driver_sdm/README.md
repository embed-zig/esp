# esp_driver_sdm

Sdkconfig binding for the ESP-IDF `esp_driver_sdm` component.

## ESP-IDF component

Maps to [`esp_driver_sdm`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/sdm.html).

## Module boundary

Typed sdkconfig definitions for the Sigma-Delta Modulation (SDM) peripheral driver. No runtime Zig API.

## Dependencies

No runtime dependencies.
