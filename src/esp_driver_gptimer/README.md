# esp_driver_gptimer

Sdkconfig binding for the ESP-IDF `esp_driver_gptimer` component.

## ESP-IDF component

Maps to [`esp_driver_gptimer`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/gptimer.html).

## Module boundary

Typed sdkconfig definitions for General Purpose Timer: IRAM placement, ISR safety, debug logging, and deprecation warning suppression. No runtime Zig API.

## Dependencies

No runtime dependencies.
