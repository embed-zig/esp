# esp_http_server

Sdkconfig binding for the ESP-IDF `esp_http_server` component.

## ESP-IDF component

Maps to [`esp_http_server`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/protocols/esp_http_server.html).

## Module boundary

Typed sdkconfig definitions for the HTTP server, including URI/header length limits, WebSocket support, purge buffer sizing, and event post timeout. No runtime Zig API.

## Dependencies

No runtime dependencies.
