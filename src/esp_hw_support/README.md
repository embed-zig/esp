# esp_hw_support

Sdkconfig binding for the ESP-IDF `esp_hw_support` component.

## ESP-IDF component

Maps to [`esp_hw_support`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/hw_support.html).

## Module boundary

Typed sdkconfig definitions for low-level hardware support, including RTC clock source selection, calibration cycles, crystal frequency, and peripheral control IRAM placement. No runtime Zig API.

## Dependencies

No runtime dependencies.
