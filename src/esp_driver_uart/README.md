# esp_driver_uart

Sdkconfig binding for the ESP-IDF `esp_driver_uart` component.

## ESP-IDF component

Maps to [`esp_driver_uart`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/uart.html).

## Module boundary

Typed sdkconfig definitions for the UART peripheral driver. No runtime Zig API.

## Dependencies

No runtime dependencies.
