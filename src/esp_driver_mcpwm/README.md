# esp_driver_mcpwm

Sdkconfig binding for the ESP-IDF `esp_driver_mcpwm` component.

## ESP-IDF component

Maps to [`esp_driver_mcpwm`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/peripherals/mcpwm.html).

## Module boundary

Typed sdkconfig definitions for Motor Control PWM: IRAM placement of control functions, ISR safety, debug logging, and deprecation warning suppression. No runtime Zig API.

## Dependencies

No runtime dependencies.
