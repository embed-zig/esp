# app_trace

Sdkconfig binding for the ESP-IDF `esp_app_trace` component.

## ESP-IDF component

Maps to [`esp_app_trace`](https://docs.espressif.com/projects/esp-idf/en/stable/esp32s3/api-reference/system/app_trace.html).

## Module boundary

Typed sdkconfig definitions for application-level tracing (JTAG, UART, USB CDC destinations and lock/priority settings). No runtime Zig API.

## Dependencies

No runtime dependencies.
