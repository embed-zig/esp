# idf_build_system

Sdkconfig binding for the ESP-IDF build system metadata.

## ESP-IDF component

Build system infrastructure component; no public runtime API documentation.

## Module boundary

Provides sdkconfig bindings for build-system-level settings: target chip/architecture selection, IDF version, toolchain choice, firmware chip ID, experimental features flag, and binary blob inclusion policy.

## Dependencies

No runtime dependencies (build-time only).
